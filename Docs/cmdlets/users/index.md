# User & Group Cmdlets

Cmdlets for managing users and groups.

| Cmdlet | Description |
|--------|-------------|
| [Get-NMMUser](Get-NMMUser.md) | Get user details |
| [Get-NMMUsers](Get-NMMUsers.md) | Search and list users |
| [Get-NMMUserMFA](Get-NMMUserMFA.md) | Get user MFA status |
| [Get-NMMGroup](Get-NMMGroup.md) | Get group details |
| [Get-NMMWorkspaceSession](Get-NMMWorkspaceSession.md) | Get workspace sessions |
