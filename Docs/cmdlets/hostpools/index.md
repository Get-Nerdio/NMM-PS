# Host Pool Cmdlets

Cmdlets for managing Azure Virtual Desktop host pools.

| Cmdlet | Description |
|--------|-------------|
| [Get-NMMHostPool](Get-NMMHostPool.md) | List host pools for an account |
| [Get-NMMHostPoolSettings](Get-NMMHostPoolSettings.md) | Get AVD settings |
| [Get-NMMHostPoolAutoscale](Get-NMMHostPoolAutoscale.md) | Get autoscale configuration |
| [Get-NMMHostPoolAD](Get-NMMHostPoolAD.md) | Get Active Directory settings |
| [Get-NMMHostPoolRDP](Get-NMMHostPoolRDP.md) | Get RDP settings |
| [Get-NMMHostPoolFSLogix](Get-NMMHostPoolFSLogix.md) | Get FSLogix configuration |
| [Get-NMMHostPoolSchedule](Get-NMMHostPoolSchedule.md) | Get scheduled jobs |
| [Get-NMMHostPoolUser](Get-NMMHostPoolUser.md) | Get assigned users |
| [Get-NMMHostPoolSession](Get-NMMHostPoolSession.md) | Get active sessions |
| [Get-NMMHostPoolVMDeployment](Get-NMMHostPoolVMDeployment.md) | Get VM deployment settings |
| [Get-NMMHostPoolTimeout](Get-NMMHostPoolTimeout.md) | Get session timeout settings |
| [Get-NMMHostPoolTag](Get-NMMHostPoolTag.md) | Get Azure resource tags |
| [New-NMMHostPool](New-NMMHostPool.md) | Create a new host pool |
| [Remove-NMMHostPool](Remove-NMMHostPool.md) | Remove a host pool |
| [Set-NMMAutoscale](Set-NMMAutoscale.md) | Configure autoscale settings |
