# Device Cmdlets (Beta API)

Cmdlets for managing Intune-enrolled devices.

!!! warning "Beta API"
    Device cmdlets use the v1-beta API and may change without notice.

| Cmdlet | Description |
|--------|-------------|
| [Get-NMMDevice](Get-NMMDevice.md) | List Intune devices |
| [Get-NMMDeviceCompliance](Get-NMMDeviceCompliance.md) | Get compliance status |
| [Get-NMMDeviceApp](Get-NMMDeviceApp.md) | Get installed applications |
| [Get-NMMDeviceAppFailure](Get-NMMDeviceAppFailure.md) | Get failed app installations |
| [Get-NMMDeviceHardware](Get-NMMDeviceHardware.md) | Get hardware information |
| [Get-NMMDeviceLAPS](Get-NMMDeviceLAPS.md) | Get local admin password |
| [Get-NMMDeviceBitLocker](Get-NMMDeviceBitLocker.md) | Get BitLocker keys |
| [Sync-NMMDevice](Sync-NMMDevice.md) | Force Intune sync |
