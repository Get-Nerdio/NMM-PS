# Authentication Cmdlets

Cmdlets for connecting to and authenticating with the NMM API.

| Cmdlet | Description |
|--------|-------------|
| [Connect-NMMApi](Connect-NMMApi.md) | Authenticate to the NMM API |
| [Get-NMMApiToken](Get-NMMApiToken.md) | Get the current cached access token |
| [New-NMMApiCertificate](New-NMMApiCertificate.md) | Create a certificate for authentication |
