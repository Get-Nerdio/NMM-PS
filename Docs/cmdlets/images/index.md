# Desktop Image Cmdlets

Cmdlets for managing golden images and templates.

| Cmdlet | Description |
|--------|-------------|
| [Get-NMMDesktopImage](Get-NMMDesktopImage.md) | List desktop images |
| [Get-NMMDesktopImageDetail](Get-NMMDesktopImageDetail.md) | Get image details |
| [Get-NMMDesktopImageLog](Get-NMMDesktopImageLog.md) | Get image change history |
| [Get-NMMDesktopImageSchedule](Get-NMMDesktopImageSchedule.md) | Get image maintenance schedules |
| [Get-NMMImageTemplate](Get-NMMImageTemplate.md) | List image templates |
